# Crashcount Feedback

This repository is the public bug-report inbox for:

- the Crashcount iPhone app
- [crashcount.nyc](https://crashcount.nyc)

Use the GitHub issue forms to report bugs and beta-test feedback.

## Report a bug

- [Report a Crashcount app bug](../../issues/new/choose)
- [Report a crashcount.nyc bug](../../issues/new/choose)

## For maintainers

Issue forms apply labels automatically so Codex can pick up structured reports quickly:

- `bug`
- `beta`
- `ready-for-codex`
- `crashcount-app`
- `crashcount-site`

The label definitions live in `.github/labels.json` and are synced by GitHub Actions.
